# Rune Grid — Game Design Document

## Концепция

**Жанр: Roguelite arena shooter.** Каждый run — с чистого листа. Game over = полный перезапуск.

Гексагональная сетка (4 клетки вдоль стороны, 37 ячеек макс.) — центральная система прогрессии внутри run. Руны дают **базовый эффект** (слабый сам по себе) и **ауру влияния** на другие руны в сетке. Правильное позиционирование создаёт синергии, превращая слабые руны в мощные комбинации.

**Ключевая фантазия:** игрок чувствует себя мастером-ювелиром, собирающим идеальную мозаику силы.

## Модель волн

- Каждая волна имеет **фиксированное количество врагов**
- Враги спавнятся поштучно (не все сразу)
- Когда все враги волны заспавнены — ждём, пока всех убьют
- После уничтожения всех врагов → Shop фаза → следующая волна
- **Нет** плавного уменьшения жизней. Жизни теряются только от урона врагов
- **Game Over** при 0 HP — полный перезапуск run

## Гексагональная сетка

Сетка — правильный гексагон с 4 клетками вдоль каждой стороны (side=4, radius=3).

```
Полная сетка (37 ячеек):

        ○ ○ ○ ○           ← edge row (4 cells)
       ○ ○ ○ ○ ○          ← 5 cells
      ○ ○ ○ ○ ○ ○         ← 6 cells
     ○ ○ ○ ○ ○ ○ ○        ← center row (7 cells)
      ○ ○ ○ ○ ○ ○         ← 6 cells
       ○ ○ ○ ○ ○          ← 5 cells
        ○ ○ ○ ○           ← edge row (4 cells)

Ring 0 (center):  1 cell
Ring 1:           6 cells   (total: 7)
Ring 2:          12 cells   (total: 19)
Ring 3:          18 cells   (total: 37)
```

Координатная система — axial (q, r). Центр = (0, 0).

### Прогрессия размера

Начинаем с внутренних колец, разблокируем ячейки за gold:
- **Старт:** Ring 0-1 (7 ячеек) — компактно, легко понять механику
- **Покупка:** отдельные ячейки Ring 2 и Ring 3 за растущую цену
- **Макс:** все 37 ячеек

### 6 направлений (hex neighbors)

```
    NW  NE
  W  ●  E
    SW  SE
```

Каждая ячейка имеет до 6 соседей. Это влияет на паттерны аур.

## Анатомия руны

Каждая руна имеет:

| Свойство | Описание |
|----------|----------|
| **Base Effect** | Прямой бонус к стату игрока (слабый) |
| **Aura Pattern** | Геометрия влияния — определяет зону для Aura Effect и Self Effect |
| **Aura Effect** | Что получают руны, попавшие в зону паттерна |
| **Self Effect** | Как руны в зоне паттерна влияют на саму эту руну |
| **Tier** | Common, Rare, Epic, Legendary (масштаб эффектов). Все тиры могут появляться с самого начала |
| **Limit** | Макс. копий на поле. Legendary=1, некоторые Common=∞ |
| **Category** | Stat rune (обычная) или Ability rune (даёт способность, макс. 3 на всю сетку) |

## Категории рун

### Stat Runes (обычные)
Модифицируют статы: +damage, +speed, +HP, +crit и т.д. Неограниченное количество на сетке (в рамках limit каждой руны).

### Ability Runes
Дают новые способности (орбитальный щит, доп. проджектайл, аура замедления и т.д.). **Максимум 3 Ability руны на всю сетку** — это жёсткий лимит. У них тоже есть base effect и aura, но главное — способность.

## Паттерны влияния (Aura Patterns) — Hex

Паттерны адаптированы под гексагональную геометрию:

```
Adjacent (6 соседей)        Line (одна из 3 осей)
                            
     X X                         X
    X * X                       X
     X X                    X X * X X
                                X
                                 X

Ring (12 клеток dist=2)     Cone (треугольник в одном направлении)

    X X X                        X X
   X . . X                     X X X
  X . * . X                   X X * 
   X . . X
    X X X

Half (полусфера)            All Same Rune
                            (anywhere on grid)
   X X X X                  
  X X X X                   S . . .
   X X *                     . S . .
    . .                     . . . S
     .
```

### Полный список паттернов

| Паттерн | Hex-описание | Кол-во целей (макс.) |
|---------|-------------|---------------------|
| **Adjacent** | 6 соседних ячеек | 6 |
| **Line** | Все клетки по одной из 3 осей (через центр руны) | до 6 |
| **Ray** | Луч в одном направлении (E, NE, NW, W, SW, SE) | до 3 |
| **Ring** | Все клетки на расстоянии ровно 2 | 12 |
| **Cone** | Треугольник в направлении (расширяется) | до 6 |
| **Half** | Все клетки по одну сторону от руны | до ~18 |
| **All Same Rune** | Все копии той же руны на сетке | variable |
| **All on Grid** | Все руны на сетке | variable |
| **Nearby (dist ≤ 2)** | Все в радиусе 2 (Adjacent + Ring) | 18 |

## Типы эффектов

### Aura Effect (руна → цели в паттерне)

| Тип | Пример | Что делает |
|-----|--------|------------|
| **Amplify** | "Adjacent: x1.5 к их base effect" | Умножает базовый эффект целей |
| **Echo** | "Line: их аура срабатывает дважды" | Удваивает ауру целей |
| **Grant** | "Adjacent: +3% damage" | Даёт плоский бонус целям |
| **Chain** | "Передаёт свой base effect руне по Ray E" | Передача эффекта по направлению |
| **Suppress** | "Ring: -50% base effect" | Негативное влияние (trade-off) |

### Self Effect (цели в паттерне → руна)

| Тип | Пример | Что делает |
|-----|--------|------------|
| **Catalyze** | "За каждую руну в Adjacent: +2% к base" | Стакающийся бонус от количества соседей |
| **Absorb** | "За каждую руну в Ring: +1% crit" | Набирает силу от окружения |
| **Mirror** | "Копирует лучший base effect из Adjacent" | Берёт эффект от соседа |

## Пример синергии (Hex)

```
Фрагмент сетки:

      [Lens] [Spark]
       [Spark]  [ ... ]

Lens (Rare):
  Base: +1% all stats
  Aura (Adjacent): Amplify x1.2 к аурам целей

Spark (Common):
  Base: +3% damage
  Aura (Adjacent): Grant +2% damage
  Self: --

Оба Spark — соседи Lens и соседи друг друга.

Spark #1 получает:
  Base:                +3% dmg
  от Spark #2 (Aura):  +2% dmg (Adjacent Grant)
  от Lens (Aura):      Amplify x1.2 к аурам Spark #1
  Итого base:          3% + 2% = 5% dmg
  Аура Spark #1 усилена Lens: раздаёт +2.4% вместо +2%

Spark #2 получает:
  Base:                +3% dmg
  от Spark #1 (Aura):  +2.4% dmg (усилена Lens)
  Итого:               5.4% dmg

vs +3% без синергий у каждого
```

```
┌──────────────────────────────────────┐
│           RUNE GRID                  │
│                                      │
│          ○ ○ ○ ○                     │
│         ○ ○ ○ ○ ○          ┌──────┐ │
│        ○ ○ ○ ○ ○ ○         │ Shop │ │
│       ○ ○ ○ ★ ○ ○ ○        │      │ │
│        ○ ○ ○ ○ ○ ○         │  R1  │ │
│         ○ ○ ○ ○ ○          │  R2  │ │
│          ○ ○ ○ ○           │  R3  │ │
│                            │      │ │
│   Abilities: [1/3]         │ Roll │ │
│                            └──────┘ │
│   Hover: [Rune details + aura vis]  │
│   Total stats: DMG +23% SPD +12%   │
└──────────────────────────────────────┘
```

При наведении на руну:
- Подсветить клетки её аура-паттерна
- Показать какие руны она усиливает и чем
- Показать кто усиливает ЕЁ

При перетаскивании руны из магазина:
- Preview: подсветка паттерна + preview итоговых статов
- Для Ability rune: показать счётчик "2/3 ability slots used"

Перестановка рун в Shop фазе свободная — можно перетаскивать руны между ячейками без ограничений.

Во время боя сетка скрыта. Можно посмотреть в режиме паузы (read-only, перестановка запрещена).

## Порядок вычислений

1. Собрать base effects всех рун
2. Применить Self Effects (Catalyze, Absorb, Mirror — руна читает окружение)
3. Применить Aura Effects: Grant, Amplify, Suppress (руна пишет в цели)
4. Применить Echo (удвоение аур — рекурсия макс. 1 уровень)
5. Суммировать финальные эффекты → статы игрока
6. Активировать Ability руны (добавляют способности)

**Важно**: Echo не рекурсивна (аура удвоенная Echo не удваивается ещё раз). Это предотвращает бесконечные петли.

## Экономика

### Получение рун
- Каждую Shop фазу предлагается 3-4 случайных руны из всех тиров
- Reroll за gold (цена: TBD)
- Все тиры доступны с самого начала, но вероятность появления зависит от тира

### Продажа
- Руну можно продать обратно за 50% от цены покупки
- Освобождает ячейку

### Разблокировка ячеек
- Старт: Ring 0-1 (7 ячеек)
- Дополнительные ячейки Ring 2 и Ring 3 покупаются за gold
- Цена растёт с каждой купленной ячейкой
